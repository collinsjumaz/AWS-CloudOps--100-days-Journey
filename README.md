## 📅 Journey Overview
This repository documents my **100‑day journey** into AWS Cloud Engineering.  
Each day I’ll add new lessons, skills, and practical exercises, creating a public learning log and portfolio.  

## 📁 Repository Structure
- `daily-logs/`: Detailed daily learning logs in Markdown format
- `README.md`: Overview and quick summaries of each day

---
## ✍️ Daily Updates
I will update this README every day with:  
- New skills learned  
- Labs completed  
- Key takeaways 

To add a new daily log:  
1. Copy `daily-logs/template.md` to `daily-logs/day-XX.md` (replace XX with the day number)  
2. Fill in the details  
3. Add a summary entry to the Day Logs section below with a link to the detailed log
